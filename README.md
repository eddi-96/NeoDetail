# 🚗 NeoDetail - Plataforma Web de E-Commerce Automotriz

Fecha: 01 de Marzo del 2026

## Estudiante
Edison Alexander Sánchez Rosero

## Objetivo del Proyecto
Desarrollar una aplicación web que permita la gestión de productos automotrices mediante servicios web, integrando autenticación, administración de productos, carrito de compras y generación de órdenes.

---

## lenguajes utilizados

- Go (Golang)
- Gin Framework
- MySQL
- HTML + CSS
- JSON (Serialización de datos)
- javascrip
- Git & GitHub

---

##  Funcionalidades Principales

1. Login de administrador (para agregar productos)
2. Gestión de productos (CRUD)
3. Visualización de tienda
4. Carrito de compras
5. Finalización de compra
6. Generación de órdenes
7. Control de sesión mediante cookies
8. Protección de rutas administrativas

---

##  Servicios Web Implementados (JSON)

1. POST /admin/login
2. GET /api/productos
3. POST /api/productos
4. PUT /api/productos/:id
5. DELETE /api/productos/:id
6. POST /api/carrito
7. GET /api/carrito
8. POST /api/checkout
